# cron jobs

## Debian cron

* `/etc/crontab` > `/etc/cron.<PERIOD>`
  * 中を読むと `/etc/cron.daily` 等の中身を `run-parts` で呼んでいる。
* `/etc/cron.d/*`

## run-parts の検索対象ルール

割と罠なので注意。

* ファイル名
  * 半角英数字
  * アンダースコア
  * ハイフン
* 実行権限
  * 当然だが必要。無いと無視される。

ファイル名にドットが含まれると無視されるので注意！！
シンボリックリンクは名前とリンク先が条件を満たしていれば OK。

run-parts 自体はかなり有用なコマンドで、`--regex` オプションを使えば
検索対象を正規表現で指定できる。
あくまでも罠なのはデフォルト条件のみである。
`/etc/crontab` の書き換えをしてよいなら変更可能だけど…。
