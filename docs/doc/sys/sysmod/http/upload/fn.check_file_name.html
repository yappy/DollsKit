fn check_file_name(name: Option<&str>) -> Result<&str, ActixError>
Expand description

ファイル名をチェックする。

  • None および空文字列は NG
  • FILE_NAME_MAX_LEN 文字まで
  • 半角英数字およびドット、ハイフン、アンダースコアのみ
  • ドットで始まらない (隠しファイルや . .. などで何かが起こらないようにする)
